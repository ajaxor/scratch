# Relic Rooms

A tiny top-down action RPG prototype inspired by the Atari Swordquest vibe, built as a single static browser page.

## Play

Open `index.html` directly from the repository, or enable GitHub Pages for this repository:

1. Go to **Settings** → **Pages**.
2. Under **Build and deployment**, choose **Deploy from a branch**.
3. Select branch **main** and folder **/** root.
4. Save.
5. Open the published Pages URL on your phone.

## Controls

### Phone

- D-pad: move
- **TOOL**: use equipped relic
- **SWAP**: cycle relics
- **READ**: read the room clue
- **RESET**: restart

### Keyboard

- Arrow keys or WASD: move
- Space or Enter: use equipped relic
- E: swap relic
- R: read clue

## Prototype Features

- Room-based top-down exploration
- Very small color palette
- Equipable relics that unlock traversal and puzzle solutions
- Vines, cracked floors, hidden walls, hook anchors, bronze doors, seals
- Enemies and reflectable projectiles
- Simple final objective using crown shards
